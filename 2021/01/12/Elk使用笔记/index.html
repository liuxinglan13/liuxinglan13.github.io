多行日志合并

^\d+-[a-zA-Z]+-\d+ \d+:\d+:\d+.\d+ 07-Aug-2020 20:15:41.950
正则匹配以 07-Aug-2020 20:15:41.950 不匹配的的行合并匹配行,

  • 例子2 zookeeper日志

    | %{TIMESTAMP_ISO8601:timestamp} [myid:\d] - %{LOGLEVEL:loglevel} (?(\s+.)+) | |
    | ———————————————————— | —- |
    | (\s+.    )+ 匹配了 WARN后的所有内容 | |

  • 正则基础

    | \S：匹配非空 | |
    | —————————————– | —- |
    | \d：匹配单个数字 | |
    | +：匹配前一个字符至少出现一次 | |
    | ?：匹配前一个字符最多出现一次 | |
    | *：匹配前一个字符出现任意次数（包括０次） | |
    | {n}：匹配妻哪一个字符出现n次 | |
